.. _Demonstrations:

デモ
=====

以下のページは、 Robot Framework の紹介や入門向けのデモプロジェクトです。

`Quick Start Guide <https://github.com/robotframework/QuickStartGuide/blob/master/QuickStart.rst>`_
    Robot Framework の最も重要な機能の紹介であると同時に、実行可能なデモになっています。

`Robot Framework demo <https://bitbucket.org/robotframework/robotdemo/wiki/Home>`_
    簡単なテストケースのサンプルです。テストライブラリを自作する方法のデモにもなっています。

`Web testing demo <https://bitbucket.org/robotframework/webdemo/wiki/Home>`_
    高水準のテストの作り方のデモです。テスト対象システムは簡単な Web ページで、 Selenium2Library_ (`和訳`_)を使っています。

`SwingLibrary demo <https://github.com/robotframework/SwingLibrary/wiki/SwingLibrary-Demo>`_
    Java の GUI アプリケーションをテストするための SwingLibrary_ のデモです。

`ATDD with Robot Framework <https://code.google.com/p/atdd-with-robot-framework>`_
    受入テスト駆動開発 (ATDD:Acceptance Test Driven Development) で Robot Framework を使う方法のデモです。

.. _Selenium2Library: http://robotframework.org/Selenium2Library/doc/Selenium2Library.html
.. _`和訳`: https://selenium2library-ja.readthedocs.org/ja/latest/
.. _SwingLibrary: http://robotframework.org/SwingLibrary/SwingLibrary-1.9.2.html
